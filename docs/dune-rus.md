Руководство по установке приложений на Dune HD STB
--------------------------------------------------

###  

### Dune HD TV-102

1.  Выбрать **Источники**.

2.  Выбрать **USB-накопитель**.

3.  Выбрать zip файл с приложением.

4.  Нажать на пульте кнопку **Enter**.

5.  Выбрать **Запустить**.
